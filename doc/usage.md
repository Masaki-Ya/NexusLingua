# NexusLingua - ユーザー向け利用ガイド

超軽量AI翻訳・アシスタントツール「**NexusLingua**」の起動方法から基本的な使い方までのマニュアルです。

---

## 1. 動作のための前提条件

本アプリは、Google AI Studio が提供する **Gemini API** を利用します。
- **Gemini API キー**が必要です。
- キーは [Google AI Studio (無料プランあり)](https://aistudio.google.com/) から無料で数秒で取得できます。

---


## 3. 設定画面（APIキー・ショートカット・最前面表示）

アプリ画面右上にある **設定アイコン（⚙️）** をクリックして設定パネルを開きます。

### APIキーの設定
1. 入力欄に取得した **Gemini API キー**（`AIzaSy...`）を貼り付けます。
2. **「保存」** ボタンをクリックします。
3. **「使用モデル」** のドロップダウンリストに、現在利用可能なAIモデルが自動で読み込まれます。
   - **推奨**: **`gemini-1.5-flash`** (制限が緩く、超高速で動作します)

### ショートカットの変更
デフォルトでは `Alt+T` と `Alt+S` に割り当てられていますが、変更が可能です。
1. 「テキスト翻訳」または「キャプチャ翻訳」の横にある **「変更」** ボタンをクリックします。
2. 画面が暗くなり「入力待ち」状態になるので、キーボードで割り当てたいキー（例: `Ctrl` + `Shift` + `C` など）を実際に押します。
3. 自動で記録され、即座に新しいショートカットが有効になります。

### 常に最前面に表示する
- **「常に最前面に表示する」** のチェックボックスをオンにすると、他のアプリ（ブラウザやWordなど）をクリックしても、NexusLinguaのウィンドウが隠れなくなります。資料を見ながら翻訳結果を確認する際に便利です。

## 4. 基本的な機能の使い方

### ① テキスト選択翻訳 (デフォルト: `Alt + T`)
画面上の文字（ブラウザ、エディタ、PDFビューアなど）をテキストとして翻訳します。

1. 翻訳したい任意の英単語や英文を、マウスのドラッグで青く選択します。
2. その状態で、キーボードの **`Alt` + `T`** を押します。
3. **自動的に選択テキストがコピー**され、NexusLingua のウィンドウが最前面にポップアップします。
4. 自動的にAIによる日本語訳が出力されます。
   - 自分で入力欄に直接キーボードでテキストを打ち込み、右下の「再実行」を押して翻訳することも可能です。

### ② 画像範囲選択翻訳 (デフォルト: `Alt + S`)
Web画像内のテキスト、電子書籍、PDFの図表、システムエラーのポップアップなど、ドラッグで文字選択できない部分を翻訳します。

1. 翻訳したい対象が画面に映っている状態で、アプリ画面内の **「📸 キャプチャ (Alt+S)」** ボタンをクリックするか、キーボードの **`Alt` + `S`** を押します。
2. **Windows標準のキャプチャ機能（Snipping Tool）が自動で起動し、画面が暗転**します。
3. マウスのカーソルが十字（`+`）に変わります。翻訳したい範囲を左クリックからドラッグ（囲む）して指定します。
   - ※ マルチモニター環境の場合、どのモニターからでも自由に選択できます。
   - ※ `Escape` キーを押すことで、いつでもキャプチャをキャンセルして元の画面に戻れます。
4. ドラッグの手を離すと、囲まれた画像が自動で切り抜かれて NexusLingua に取り込まれます。
5. AIが画像をそのままスキャンし、画像内の外国語を日本語に翻訳して出力します。

---

## 5. モード（出力形式）の切り替え

インプットされたテキストや画像に対して、AIのアウトプット形式を3つのタブで切り替えることができます。

- **「翻訳」タブ**: 
  意味を崩さず、自然で読みやすい日本語の文章へ翻訳します。
- **「要約」タブ**: 
  翻訳した文章の重要なポイントを抽出し、日本語の箇条書きでコンパクトにまとめます。長文をざっくり理解したいときに便利です。
- **「解説」タブ**: 
  入力内容がプログラミングコードや専門用語の場合、その意味や処理フロー、注意点などを詳細に解説します。

*※一度結果を取得したタブの情報は**キャッシュ**されます。そのため、同じ画像やテキストのままでタブを行き来しても、無駄にAPIを消費することなく、一瞬で過去の結果が表示されます。手動で強制的に翻訳し直したい場合は右下の「再実行」ボタンを押してください。*
