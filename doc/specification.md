# NexusLingua - 技術仕様書

Windows上で軽快に動作し、選択テキストや画面上の画像から一瞬でAI翻訳を行うアシスタントアプリ「**NexusLingua**」の技術仕様書です。

---

## 1. アプリケーション概要

- **アプリケーション名**: NexusLingua（ネクサス・リングア）
- **対象OS**: Windows 10 / 11
- **コンセプト**: 「軽快・高速・スマート」
  従来のブラウザ切り替えや重いデスクトップアプリのストレスを解消し、任意の画面上でショートカットキー一発で起動・翻訳できるAIアシスタント。

---

| レイヤー | 技術 / ライブラリ | 役割・メリット |
| :--- | :--- | :--- |
| **コアフレームワーク** | **Tauri (v2)** | アプリの軽量化（Rustベースのバックエンド、WebViewによるUI）。メモリ消費量約30〜50MBの高速起動を実現。 |
| **バックエンド (Rust)** | `enigo` (v0.2) | キーボード送信（`Ctrl + C`）のシミュレーションによるテキスト自動コピー。 |

## 3. 主要機能仕様

### A. テキスト選択翻訳 (デフォルト: `Alt + T`)
画面上の任意のテキストを選択した状態でショートカットキーを押すだけで、即座に翻訳結果を表示します。

1. **イベント検知**: Rust側でグローバルに `Alt + T` を監視。
2. **コピーエミュレーション**: アクティブな他アプリに対して `Ctrl + C` キー送信をエミュレートし、選択文字列を強制的にシステムクリップボードへコピー。
3. **クリップボード取得**: `tauri-plugin-clipboard-manager` を用いて、テキストを即時取得。
4. **表示・フォーカス**: メインウィンドウを背後から最前面へ表示・アクティブ化。
5. **APIリクエスト**: フロントエンドにテキストを送信し、Gemini APIを叩いて翻訳（または要約・解説）を実行。

### B. 画像範囲選択翻訳 (デフォルト: `Alt + S` / UIボタン)
画面上のテキスト化できない文字（PDF、画像、システムエラー画面など）を、ドラッグ範囲指定で直接マルチモーダル翻訳します。

1. **イベント検知**: Rust側でグローバルに `Alt + S` を監視、またはフロントエンドのボタンからコマンド呼び出し。
2. **Snipping Tool 起動**: `enigo` を用いて、OS標準のキャプチャショートカット `Win + Shift + S` をバックグラウンドでエミュレート送信。
3. **OS標準キャプチャ**: 画面が暗転し、マルチモニター環境にも完全対応したOSネイティブの領域選択モードに移行。
4. **クリップボード監視**: Rust側でバックグラウンドスレッドを立ち上げ、最大30秒間クリップボードをポーリング監視。ユーザーが範囲選択を終えてクリップボードに画像がコピーされた瞬間を検知。
5. **画像翻訳リクエスト**: `tauri-plugin-clipboard-manager` を用いて画像を取得・Base64化し、メインウィンドウへ画像を転送。Gemini APIに対して画像を直接入力として送信。OCRを挟むことなく、コンテキストと文字の配置を理解した自然な翻訳を出力。

### C. Geminiモデルの動的取得・フォールバック
ユーザーのアカウントやAPIキーによって使用可能なモデルが異なる問題をクリアするための安全仕様です。

1. **ListModels APIの呼び出し**: アプリ起動時およびAPIキーの保存時に、`https://generativelanguage.googleapis.com/v1beta/models?key={apiKey}` を実行。
2. **モデルフィルタリング**: 取得したモデルの中から、テキスト生成（`generateContent`）に対応したモデルのみを抽出。
3. **ドロップダウン同期**: 設定画面（⚙️）の「使用モデル」の選択肢を、APIキーで**本当に使用可能なモデル一覧**で動的に上書き。無効なモデル名指定による 404 エラーを完全に防ぎます。
4. **フォールバック**: ネットワークエラー等でリストの取得に失敗した場合は、標準の3モデル（`gemini-1.5-flash`, `gemini-2.0-flash`, `gemini-1.5-pro`）をフォールバックとして提供。

### D. AI翻訳アシスト（モード切り替え）
UI上部のタブを切り替えることで、同一のインプット（テキストまたは画像）に対してAIに異なる処理を指示できます。
- **翻訳**: 直訳にこだわらず、日本語として自然な表現にリライトします。
- **要約**: 箇条書きを用いて、長文の要点を素早く把握できるようにまとめます。
- **解説**: コードの意図や専門用語の意味など、詳細な解説を付与します。

### E. ショートカットキーの動的カスタマイズ
ユーザーが使いやすいように、グローバルショートカットキーを自由に変更可能です。
1. **設定UI (モーダル)**: フロントエンドでキー入力（例: `Ctrl+Shift+C`）をフックし、`tauri-plugin-global-shortcut` 互換の文字列に変換。
2. **Rustコマンド**: `update_shortcuts` コマンドを呼び出し、既存のショートカットを `unregister_all()` してから新しいキーを再登録します。

### F. 常に最前面表示 (Always on Top)
他の作業をしながらでもアプリウィンドウが背後に隠れないようにします。
- Tauriフロントエンドのセキュリティ制約を回避するため、Rust側の `set_window_always_on_top` コマンドを使用して、OSネイティブの権限で確実に最前面に固定します。

### G. AIレスポンスのキャッシュ最適化
無駄なAPI呼び出しによるトークン消費を防ぎます。
- **タブ切り替え時の再利用**: 入力内容（テキスト/画像）が変わらない限り、「翻訳」「要約」「解説」の各モードの初回APIレスポンスをフロントエンドのメモリ上にキャッシュ。タブを切り替えてもAPI通信なしで瞬時に結果を表示します。
